Game Development

Ever occuring change

Since the first video-game was published, the production of video games has drastically changed. What started out as university projects on analog computers is now an industry estimated to be worth 90 billion dollars (source: We PC) using countless software and proprietary hardware.

It has become easier than ever for a 1 man team to make a video-game thanks to the many tools available on the market. Whether it is a one man studio or a huge company, the basic workflow remains the same. A major deciding factor on the workflow will be if the game is 2D or 3D. We will focus on 3D in this page. The processes may vary according to the project but they usually follow the following workflow:

Concept Art

"a form of illustration used to convey an idea for use in films, video games, animation, comic books, or other media before it is put into the final product" -Wikipedia

Art Side Workflow

Modeling

3D modeling is the process of developing a mathematical coordinate-based representation of any surface of an object (inanimate or living) in three dimensions via specialized software

Texturing

Texturing is a method for defining high frequency detail, surface texture, or color information on a computer-generated graphic or 3D model

Rigging

Rigging is a technique in computer animation in which a character (or other articulated object) is represented in two parts: a surface representation used to draw the character (called the mesh or skin) and a hierarchical set of interconnected parts (called bones, and collectively forming the skeleton or rig), a virtual armature used to animate (pose and keyframe) the mesh

Animating

Animating is a process used to create movement to models such as characters. It involves keyframing armatures to certain positions

Technical Aspects

Game Engines

A game engine is conceptually the core software necessary for a game program to properly run. In many cases, game engines provide a suite of visual development tools in addition to reusable software components. These tools are generally provided in an integrated development environment to enable simplified, rapid development of games in a data-driven manner.

Programming

Game programming, is the software development of video games. Game programming requires substantial skill in software engineering and computer programming in a given language, as well as specialization in one or more of the following areas: simulation, computer graphics, artificial intelligence, physics, audio programming, and input

Networking

Networking involves setting up infrastructure and code for users to remotely connect and play together
